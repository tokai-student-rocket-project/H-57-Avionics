## 概要

### 目的

- 酸化剤供給用バルブと主流路用バルブの制御を行う
- 位置情報を取得しロケットの現在地を地上に送信する

### 特徴

- 使用するマイコンボードにArduinoを採用
  →　プログラミング初学者でも比較的作成が容易
  → 拡張性・運用性・保守性が向上
- LoRaWANの搭載
  - 簡単なプログラムで見通し10km以上の通信が可能
